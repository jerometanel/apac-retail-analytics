## What This Project Demonstrates

A complete end-to-end analytics project built on a synthetic 
APAC retail dataset covering 800 clients, 3,053 transactions, 
8 markets, 12 product categories, and 2 years of data.

| Layer | Tool | What I built |
|---|---|---|
| Data preparation | SQL | 5 diagnostic queries |
| Analysis | Python / pandas | Purchase path study |
| Visualisation | Power BI / charts | 4-chart dashboard |
| Insights | Written summary | Executive findings |

---

## Key Business Finding

Handbag purchase is the strongest predictor of client 
segment upgrade — 62% of Regular clients who purchased 
a Handbag moved to Elite within 6 months.
